## Решение Тестового задания по автоматизированной сборке SQLite

Сборка библиотеки SQLite под Windows и Linux, которую мы загружаем по ссылке + воспроизводимое окружение через Docker, Vagrant, Ansible и Conan.

## Собрали SQLite под Windows (MSVC) и Linux (gcc/WSL)  

В рамках работы столкнулись со следующими задачами:  

- Собрана **shared**-библиотека (`.dll` / `.so`)
- Настроен **RPATH** (`$ORIGIN`), чтобы исполняемый файл находил библиотеку рядом с собой
- Написана тестовая программа на C → проверка `sqlite3_libversion()`
- Решена проблема с кодировкой логов в MSBuild (переход на английский + `/v:m`)

## Написали лёгкий Dockerfile на alpine  

- Сборка `libsqlite3.so` одной командой внутри контейнера
- Монтирование текущей директории через volume → результат сразу на хосте
- Логирование компиляции через `tee`
- Проверка работоспособности осуществляется: `docker run … gcc --version`

## Сделали всю работу в связке Vagrant + VirtualBox + Debian → команда vagrant up поднимает всё  

- Ansible (dockerplaybook.yml) ставит Docker → собирает SQLite в контейнере → делает задание 8 через Conan
- task8: main.cpp + fmt через conan + CMake
- Всё идемпотентно (creates, проверка наличия файлов)
- Полезные теги для отладки / выборочного запуска:
- - ansible-playbook dockerplaybook.yml --tags install_docker
- - ansible-playbook dockerplaybook.yml --tags Sqlite
- - ansible-playbook dockerplaybook.yml --tags Conan
